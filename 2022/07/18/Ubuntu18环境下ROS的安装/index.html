Ubuntu18环境下ROS的安装
Jul 18 2022

Ubuntu系统更改科大源

1.1 软件配置

打开“软件和更新”，确保勾选”restricted”， “universe，” 和 “multiverse”

1.2 更改科大源

在“下载自：”中选择科大源，关闭并更新(清华源在安装ROS时会报错)

设置key

sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80'   
--recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

安装

3.1 更新apt

sudo apt update

3.2 安装ROS

sudo apt install ros-melodic-desktop-full

Ubuntu18对应的ROS版本为melodic，注意区分版本！

配置环境变量

配置环境变量，以在任意终端中使用ROS

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc source ~/.bashrc

安装构建依赖

5.1 安装构建依赖的相关工具

安装工具:

sudo apt install python-rosdep python-rosinstall python-rosinstall-generator python-wstool build-essential

安装rosdep

sudo apt install python-rosdep

5.2 解决境外资源屏蔽问题

  • 列表项列表项修改/etc/hosts文件：
sudo gedit /etc/hosts

添加域名与IP映射内容：

199.232.28.133  raw.githubusercontent.com
  • 使用资源代理加速rosdep对GithubRaw的访问：
sudo gedit /usr/lib/python2.7/dist-packages/rosdep2/sources_list.py

添加:

url="https://ghproxy.com/"+url
  • 修改:
sudo gedit /usr/lib/python2.7/dist-packages/rosdistro/__init__.py

将DEFAULT_INDEX_URL修改为：

/usr/lib/python2.7/dist-packages/rosdistro/__init__.py
  • 添加：
sudo gedit /usr/lib/python2.7/dist-packages/rosdep2/gbpdistro_support.py

在第205行添加

gbpdistro_url = "https://ghproxy.com/" + gbpdistro_url
  • 同样修改以下文件：
/usr/lib/python2.7/dist-packages/rosdep2/gbpdistro_support.py
/usr/lib/python2.7/dist-packages/rosdep2/sources_list.py
/usr/lib/python2.7/dist-packages/rosdep2/rep3.py
/usr/lib/python2.7/dist-packages/rosdistro/manifest_provider/github.py

5.3 初始化rosdep

sudo rosdep init  
rosdep update

测试ROS

  1. 启动新终端，执行：roscore
  2. 启动新终端，执行:rosrun turtlesim turtlesim_node
    弹出图形界面
  3. 启动新终端，执行：rosrun turtlesim turtle_teleop_key
    控制乌龟运动
