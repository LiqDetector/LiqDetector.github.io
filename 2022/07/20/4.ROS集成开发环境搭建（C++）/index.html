ROS集成开发环境搭建
Jul 20 2022

下载与安装VScode

1.1 下载VScode

官网下载地址： https://code.visualstudio.com/

下载时若速度过慢，可将链接中的源地址改为国内地址：vscode.cdn.azure.cn

1.2 安装VScode

  • 双击安装
  • 或命令行安装：sudo dpkg -i xxxx.deb

1.3 卸载VScode（可选）

sudo dpkg --purge code

VScode插件安装

  • C/C++
  • CMake Tools
  • Python
  • ROS
  • Code Runner

VScode配置

3.1 创建ROS工作空间

mkdir -p xxx_ws/src
cd xxx_ws
catkin_make

3.2 启动VScode

cd xxx_ws
code .

3.3 设置快捷键

  1. 快捷键 ctrl + shift + B 调用编译，选择:catkin_make:build
  2. 新建.vscode/tasks.json文件
    "version": "2.0.0",
    "tasks": [
        {
            "label": "catkin_make:debug", //代表提示的描述性信息
            "type": "shell",  //可以选择shell或者process,如果是shell代码是在shell里面运行一个命令，如果是process代表作为一个进程来运行
            "command": "catkin_make",//这个是我们需要运行的命令
            "args": [],//如果需要在命令后面加一些后缀，可以写在这里，比如-DCATKIN_WHITELIST_PACKAGES=“pac1;pac2”
            "group": {"kind":"build","isDefault":true},
            "presentation": {
                "reveal": "always"//可选always或者silence，代表是否输出信息
            },
            "problemMatcher": "$msCompile"
        }
    ]
}

3.4 创建ROS功能包

  1. 选定目录中的 src –> 右键 –> create catkin package
  2. 设置包名
  3. 添加依赖

3.5 引入C++

在功能包的src下新建cpp文件

#include "ros/ros.h"

int main(int argc, char *argv[])
{
    setlocale(LC_ALL,"");
    //执行节点初始化
    ros::init(argc,argv,"HelloVSCode");

    //输出日志
    ROS_INFO("Hello VSCode!");
    return 0;
}

PS：
修改.vscode/c_cpp_properties.json
设置为”cppStandard”: “c++17”

3.6 配置CMakeLists.txt

add_executable(节点名称
  src/C++源文件名.cpp
)
target_link_libraries(节点名称
  ${catkin_LIBRARIES}
)

3.7 编译、执行

  1. 编译：ctrl + shift + B
  2. 新建一个终端，执行: roscore
  3. 新建一个终端，执行: source ./devel/setup.bash
  4. rosrun 包名 节点名

优化launch文件

  1. 选定功能包右键 –> 添加launch文件夹
  2. 选定launch文件夹右键 –> 添加launch文件
  3. 编辑 launch 文件内容
<launch>
    <node pkg="helloworld" type="demo_hello" name="hello" output="screen" />
    <node pkg="turtlesim" type="turtlesim_node" name="t1"/>
    <node pkg="turtlesim" type="turtle_teleop_key" name="key1" />
</launch>
  • node：包含的某节点
  • pkg：功能包
  • type：被运行节点文件
  • name：节点名称
  • output：日志输出目标
  1. 运行launch文件
    roslaunch 包名 launch文件名
